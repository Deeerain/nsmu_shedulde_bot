# Телеграм-бот для получения расписания занятий Сгму

## Технологии:
- Python
- python-telegram-bot
- sqlalchemy
- alimbic
- requests
- Bs4(lxml)

## Запуск(Windows):
`# git clone https://github.com/Deeerain/nsmu_shedulde_bot.git`

`# cd nsmu_shedulde_bot`

`# python -m venv env`

`# .\env\Scripts\activate`

`pip -r requirements.txt`

`# python main.py run`
